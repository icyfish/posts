## maupassant - colors


- 链接颜色: #42b983

- 评论框颜色: #15ac67; &hover: #42b983

- selection有两种方案: 

  现在正在用的:
  ```css
  ::selection {
    background: #555;
    color: #fff; 
  }
  ```

  这个是Mac系统高亮的其中一种配色(Green), 觉得很好看:
  ```css
  ::selection {
    background: #c3f4b2;
    color: #555; 
  }
  ```

- code:

  ```css
  code {
    color: #c7254e;
    border-radius: 4px; 
  }
  ```
